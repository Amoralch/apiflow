# ContentFlow

Lightweight content scheduling and analytics dashboard for multi-platform social media management.

## Features

- Unified posting queue across platforms
- - Engagement analytics with weekly reports
  - - Media asset pipeline (resize, compress, watermark)
    - - Webhook-based notifications for post status
      - - Role-based access for team collaboration
       
        - ## Stack
       
        - - Python 3.12+
          - - FastAPI
            - - SQLite (default) / PostgreSQL
              - - Jinja2 templates
                - - HTMX for interactive UI
